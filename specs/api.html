post /questions

Positionne une question sur le jeu. L'ID doit être vide pour cette requête

  • Request
  • Response

Query Parameters

  • authToken: required (string)

    Le jeton d'authentification.

  • question: required (object)

    La question à insérer.

    • id: required (integer)
    • question: required (string)
    • rep1: required (string)
    • rep2: required (string)
    • rep3: required (string)
    • rep4: required (string)
    • bonneRep: required (integer)
    • position: required (array of integer)
    • rayon: required (integer)

HTTP status code 200

Code retourné si aucune erreur n'a survenue lors de l'ajout de la question.

HTTP status code 500

Code retourné si une erreur a été rencontrée lors de l'ajout de la question.
